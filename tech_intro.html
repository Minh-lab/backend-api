2. Backend

2.1. PHP 8.2 + Laravel 12

Laravel là PHP framework theo mô hình MVC (Model – View – Controller).

Lý do chọn Chi tiết
Năng suất cao Eloquent ORM, Route, Middleware giúp code nhanh, dễ đọc
Hệ sinh thái phong phú Có sẵn package xử lý xác thực, queue, mail, v.v.
Dễ học Tài liệu phong phú, cộng đồng lớn, phù hợp nhóm sinh viên
Phiên bản mới nhất Laravel 12 + PHP 8.2 với hiệu năng cải thiện đáng kể

2.2. Laravel Sanctum (Xác thực)

Laravel Sanctum là package xác thực chính thức của Laravel, được dùng để cấp phát và quản lý API token.

  • Nhẹ và đơn giản: Được thiết kế riêng cho SPA và mobile API.
  • Hỗ trợ đa hình (Polymorphic Token): Lưu token kèm tokenable_type và tokenable_id, tự động nhận diện đúng model (Admin / Sinh viên / Giảng viên / Doanh nghiệp) mà không cần bảng users chung.
  • Tích hợp sẵn trong Laravel: Không cần cài thêm thư viện bên ngoài.

2.3. MySQL (Cơ sở dữ liệu)

  • Hệ quản trị CSDL quan hệ phổ biến nhất, phù hợp dữ liệu có cấu trúc rõ ràng (sinh viên, đề tài, thực tập...).
  • Tích hợp tốt với Laravel Eloquent ORM.

3. Frontend

3.1. React 19 + Vite 7

React là thư viện JavaScript của Meta để xây dựng giao diện theo mô hình Component.

Lý do chọn Chi tiết
Component-based Chia UI thành các thành phần tái sử dụng, dễ bảo trì và mở rộng
Cộng đồng lớn nhất Tài nguyên học tập, thư viện hỗ trợ phong phú nhất hiện tại
Vite thay CRA Hot-reload cực nhanh nhờ ES Module, build production nhẹ và tối ưu hơn

3.2. TailwindCSS 4

  • Viết style trực tiếp trong JSX bằng utility class — không cần tạo file .css riêng.
  • Đảm bảo nhất quán design system, dễ responsive trên mọi thiết bị.
  • v4 — phiên bản mới nhất, nhanh hơn đáng kể so với v3.

3.3. shadcn/ui (UI Components)

  • Bộ component đẹp, chuẩn accessibility, xây dựng trên Radix UI.
  • Code component được copy thẳng vào project → toàn quyền tùy chỉnh, không bị lock-in bởi npm package.
